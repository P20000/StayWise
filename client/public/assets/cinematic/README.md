# StayWise.ai — Cinematic Video Hero Assets Directory

Place your high-resolution architectural loop (`staywise-architectural-tour.mp4`) and static reduced-motion / mobile fallback poster (`final-frame-poster.webp`) inside this directory (`client/public/assets/cinematic/`).

### Production Guidelines (`Rule #9`)
- **Video Format**: MP4 (H.264 / AV1) encoded with high bitrate and zero audio track (`muted`).
- **Poster Format**: WebP (`800x600` mobile / `1920x1080` desktop) optimized under 150 KB.
