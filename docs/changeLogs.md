date: 2026-07-09
what changed : Refactored documentation sync architecture to create separate standalone sub-pages under a parent page titled 'StayWise Documentation' instead of writing to the prompts database, and cleared accidental doc entries from the prompts database.
what files changed : `scripts/sync_docs_to_notion.py`, `scripts/sync_docs_from_notion.py`
why changed : To prevent cluttering the prompts database which is reserved for user-written prompts, and to store all documentation files cleanly as separate individual Notion pages.
---
date: 2026-07-09
what changed : Integrated the official Notion CLI (`ntn`) into our documentation sync scripts to replace custom manual HTTP API block parsing, mapping database properties dynamically.
what files changed : `scripts/sync_docs_to_notion.py`, `scripts/sync_docs_from_notion.py`
why changed : To increase synchronization robustness, reduce code complexity, leverage the Notion CLI's official markdown parsing, and use session-based authentication instead of hardcoded tokens.
